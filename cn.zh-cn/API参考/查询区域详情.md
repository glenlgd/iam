# 查询区域详情<a name="zh-cn_topic_0067148044"></a>

## 功能介绍<a name="s1fea94fc86654d20a4264b290de6701b"></a>

该接口用于查询区域详情。

该接口可以使用全局区域的域名和其他区域的域名调用，全局区域的域名为iam.myhuaweicloud.com。

-   URI格式

    GET /v3/regions/\{region\_id\}
