# 创建用户组<a name="zh-cn_topic_0057845650"></a>

## 功能介绍<a name="section16448035101644"></a>

该接口提供创建用户组的功能。

该接口可以使用全局区域的域名和其他区域的域名调用，全局区域的域名为iam.myhuaweicloud.com。

## URI<a name="section60989708101648"></a>

URI格式

POST /v3/groups
