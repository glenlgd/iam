# 查询用户组详情<a name="zh-cn_topic_0057845618"></a>

## 功能介绍<a name="section374725841447"></a>

该接口提供查询用户组详细信息的功能。

该接口可以使用全局区域的域名和其他区域的域名调用，全局区域的域名为iam.myhuaweicloud.com。

## URI<a name="section37905744151541"></a>

-   URI格式

    GET /v3/groups/\{group\_id\}
