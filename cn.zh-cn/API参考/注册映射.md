# 注册映射<a name="zh-cn_topic_0057845590"></a>

## 功能介绍<a name="section5391497194352"></a>

该接口用于注册一个映射。

该接口可以使用全局区域的域名和其他区域的域名调用，全局区域的域名为iam.myhuaweicloud.com。

## URI<a name="section4532847794352"></a>

-   URI格式

    PUT /v3/OS-FEDERATION/mappings/\{id\}
