# 删除用户<a name="zh-cn_topic_0057845630"></a>

## 功能介绍<a name="s5888597838b0425a92e3419fb766c7f5"></a>

该接口用于删除指定用户。

该接口可以使用全局区域的域名和其他区域的域名调用，全局区域的域名为iam.myhuaweicloud.com。

-   URI格式

    DELETE /v3/users/\{user\_id\}
