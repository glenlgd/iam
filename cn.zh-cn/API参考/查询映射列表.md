# 查询映射列表<a name="zh-cn_topic_0057845567"></a>

## 功能介绍<a name="section35784939102440"></a>

该接口用于查询映射列表信息。

该接口可以使用全局区域的域名和其他区域的域名调用，全局区域的域名为iam.myhuaweicloud.com。

## URI<a name="section48981904102440"></a>

URI格式

GET /v3/OS-FEDERATION/mappings
